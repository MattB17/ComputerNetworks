UDP Client¶

The UDPClient class implements a client using the UDP Protocol.

class Networking.UDPPinger.UDPClient(timeout)[source]¶

Implements a UDP Client that waits timeout seconds for responses.

Parameters:

timeout: int

The number of seconds that the client waits for a response to a message it has sent.

Attributes

_client_socket: socket.socket The socket that the client interacts with.

Methods

get_socket_timeout() Retrieves the timeout set on the client’s socket.
get_socket_timeout()[source]¶

Retrieves the timeout set on the client’s socket.

Returns:

int

An integer corresponding to the number of seconds that the client’s socket waits for a response.
