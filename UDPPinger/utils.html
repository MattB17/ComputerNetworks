Utility Functions¶

A collection of utility functions used to implement the UDP protocol for a UDP Pinger.

Networking.UDPPinger.utils.create_pong(message_lst)[source]¶

Modifies message_lst to create the components of a pong message.

That is, a list is constructed representing a pong (a response) to message_lst.

Parameters:

message_lst: list

A list representing the message to be replied to.
Returns:

list

A list with the components of a pong message which represents the reply to message_lst.
Networking.UDPPinger.utils.process_message(message)[source]¶

Processes the message given by message.

message is a message received by a UDP node. The message is processed in order to derive a response message.

Parameters:

message: str

A string representing a message received by a UDP node.
Returns:

str

A string representing the node’s response to message.
Networking.UDPPinger.utils.receive_and_decode_message(socket, port)[source]¶

Receives and decodes a message received through socket at port.

Parameters:

socket: socket.socket

The socket at which the message is received

port: int

An integer representing the port number at which the message is received.
Returns:

string

A string representing the message received by socket, after it has been decoded.
Networking.UDPPinger.utils.send_ping_through_socket(ping_count, socket, dest_host, dest_port)[source]¶

Sends a ping message of count ping_count through socket.

The ping message is sent through socket to the destination given by dest_host and dest_port.

Parameters:

ping_count: int

An integer representing the ping count set in the sent message.

socket: socket.socket

The socket through which the ping message is sent.

dest_host: str

A string identifying the host name marking the message’s destination.

dest_port: int

An integer identifying the port number marking the message’s destination.
Returns:

None
Networking.UDPPinger.utils.validate_proportion(proportion)[source]¶

Validates that proportion corresponds to a valid proportion.

A proportion is valid if it is in the range [0, 1].

Parameters:

proportion: float

The proportion to be validated.
Returns:

None
Raises:

InvalidProportion

If proportion is not valid.
Networking.UDPPinger.utils.will_drop_message(reliability)[source]¶

Determines if a message will be dropped by a UDP node.

A message is deemed dropped if it is either not successfully received or is not successfully sent.

Parameters:

reliability: float

A float representing the reliability of a UDP node. This is a number between 0 and 1 representing the long run proportion of messages dropped by the node.
Returns:

bool

True if the message will be dropped. Otherwise, False.
